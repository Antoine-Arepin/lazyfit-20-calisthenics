# LazyFit 20 - Calisthenics Workout App

A minimalist calisthenics workout app designed to help you complete 20-exercise bodyweight workouts. Built with React for seamless web and mobile experience.

### Current Version (1.0)
- **20 Core Exercises**: Complete workout routine with exercises like Jumping Jacks, Push-ups, Squats, Planks, Lunges, and more
- **Exercise Timer**: Built-in countdown timer for each exercise
- **Rest Periods**: Automatic rest periods between exercises
- **Workout History**: Track your completed workouts
- **Progress Tracking**: View your workout streak and completion statistics
- **Lazy Mode**: Option to adjust exercise duration for easier sessions
- **Responsive Design**: Works on desktop and mobile devices
- **Dark Theme**: Eye-friendly dark interface

## üöÄ Getting Started

### Quick Start
1. Open `index.html` in a modern web browser
2. Click "Start Workout" to begin your 20-exercise routine
3. Follow the on-screen timer and instructions
4. Take rest breaks between exercises
5. Complete all 20 exercises to finish your workout

### Browser Requirements
- Modern browsers (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- LocalStorage support for workout history

1. **Home Screen**: View your current streak and start a new workout

4. **Settings**: Customize lazy mode duration and preferences
